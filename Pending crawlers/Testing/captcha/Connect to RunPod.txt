Connect to RunPod
There are many ways to interact with RunPod:

Web interface
CLI
SDKs
Web interface
Create an account and then log into the web interface at the following address: runpod.io/console/login.

CLI
You can use RunPod's CLI runpodctl to manage Pods and for development.

All Pods come with runpodctl installed with a Pod-scoped API key, which makes managing your Pods easier through the command line.

SDKs
RunPod provides SDKs for the following programming languages:

GraphQL
JavaScript
Python
